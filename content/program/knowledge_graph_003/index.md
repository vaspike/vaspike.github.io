## 版本

如果不想和不久前的我一样痛苦,版本就跟我保持一致

| DB或依赖名称           | version | 备注                           |
|-------------------|---------|------------------------------|
| Neo4j             | 5.24.0  | 版本较新,够用                      |
| spring-boot       | 3.1.1   | 高一点SDN就有bug;低到2.*则SDN的语法完全变样 |
| spring-data-neo4j | 7.3.1   | 重头戏来了,必须锁住版本为7.3.1           |
| JDK               | 17      | 能上17就用17                     |
|                   |         |                              |

**不要轻易尝试更换这里的版本!**
